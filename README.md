# RB Website

A modern, interactive website featuring an image gallery and dual calculator functionality.

## Features

### Image Gallery
- Responsive grid layout displaying 11 images
- Click images to toggle between color and grayscale
- Smooth hover effects and transitions

### Dual Calculators
- Two independent calculators starting at 8000
- Operations available:
  - `-1000` (subtract 1000)
  - `-100` (subtract 100) 
  - `-50` (subtract 50)
  - `/2` (divide by 2, rounds up)
- Reset button to restore both calculators to 8000
- Calculator-style displays with green text on black background

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, Transitions)
- JavaScript (ES6)
- Responsive Design
